This project is a personal "About Me" webpage that I built for my Web Dev I class. This project showcases my biography, some hobbies, a photo gallery of those hobbies, my short blog, a video for my midterm project, and a contact form. This website is fully accesible across mobile, tablet, and desktop screen sizes and this website is deployed live from netlify.

Description
This project was made to help me practice my web dev skills and help me build it from the ground up.
The following section is gonna be a breakdown of what I have learned and applied from each section:

-HTML Structures: throughout my project I used elements like <header>, <nav>, <main>, <section>, <article>, <figure>, <footer>, and <form> to keep my website looking nice and organized

-Nav Bar: It was built using flex box.

-Hero Section: This section displays my avatar imagew with my name and it used a centered text layout

-Biography & Hobbies: This section uses a lists, paragraph text, and CSS selectors like nth-last-child to alternate my list colors

-Photo Gallery: it uses <figure> and <figcaption> elements with responsive images (max-width: 100%). This is to adjust the images on different screen sizes.

-Litearature Section: THis section has a styled drop cap using the ::first-letter pseudo-element.

-Embedded Video: This section uses embedding from a youtube iframe and includes proper accessibility features.

-Contact Form: It was built with <fieldset>, <legend>, text inputs, email, telephone, a <select> dropdown, a <textarea>, radio buttons, and a submit button. It is laid out using a CSS Grid and the form is connected to the netlify form section for submissions.
